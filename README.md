# Mehary Aregu — Personal Portfolio

A personal portfolio website for **Mehary Aregu**, Software Engineer & ERP Analyst based in Addis Ababa, Ethiopia.

🌐 **Live site:** [https://techpro369.github.io/meharyaregu.github.io/](https://techpro369.github.io/meharyaregu.github.io/)

---

## About

This is a single-file, dependency-free portfolio website highlighting Mehary's professional experience, education, technical skills, and selected projects. The design is modern, minimal, and bilingual — supporting both **English** and **አማርኛ (Amharic)**.

## Features

- **Modern hero section** with a continuously waving 👋, an animated typewriter cycling through professional roles, and clean call-to-action buttons
- **About section** with a profile photo, "Open to Work" badge, professional summary, and animated count-up impact stats (years of experience, degrees earned, students taught)
- **Experience cards** detailing current and past roles
- **Education cards** for both M.Sc. and B.Sc. degrees
- **Bento-style skills grid** organized by Languages, Frameworks, Tools, and Domains
- **Project cards** showcasing selected enterprise software work
- **Contact form** that opens the visitor's email client with a pre-formatted message
- **Resume modal** — a one-click in-page resume with print support, opened from the hero section
- **Bilingual UI** — full English / Amharic translation toggle in the navigation
- **Responsive design** — desktop nav with full link list; mobile hamburger that morphs into an X
- **Accessibility** — semantic HTML, `aria-*` attributes, keyboard-accessible modal, reduced-motion support

## Tech Stack

- **HTML5** — semantic, single-file structure
- **CSS3** — custom properties, grid, flexbox, keyframe animations
- **Vanilla JavaScript** — no frameworks, no build step
- **Google Fonts** — Bricolage Grotesque (display), Manrope (body), JetBrains Mono (UI labels), Noto Sans Ethiopic (Amharic)
- **Hosted on** GitHub Pages

## Local Development

This site is deliberately built as a single `index.html` file with no build step. To work on it locally:

```bash
# Clone the repo
git clone https://github.com/techpro369/meharyaregu.github.io.git
cd meharyaregu.github.io

# Open in any browser
open index.html
```

Or simply double-click `index.html` from the file system.

To make edits, open `index.html` in any code editor and modify the HTML, CSS (inside the `<style>` tag), or JavaScript (inside the `<script>` tag).

## Deployment

2. GitHub Pages rebuilds the site within ~1 minute
